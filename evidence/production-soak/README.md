# Production Soak Evidence

This directory is for Evidence Output generated by real EvoPilot runs. It is intentionally separate from Product Kit assets under `examples/`.

## What Belongs Here

- dated soak reports
- release decision JSON
- loop runtime transcripts
- Source-to-GA screenshots
- trace and replay summaries
- audit evidence

## What Does Not Belong Here

- reusable demo project definitions
- GitHub workflow templates
- ExecutorAdapter example source
- permanent onboarding documentation
- secrets, tokens, `.env` files, or private logs

## Suggested Layout

```text
evidence/production-soak/YYYY-MM-DD-source-to-ga-demo/
  report.md
  release-decision.json
  loops.json
  screenshots/
```
